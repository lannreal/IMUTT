## 🔮 Tentang Proyek Ini
**NEXUS AI** (berkas utama: `dp.js`) adalah sebuah skrip *reverse-engineering* tingkat dewa dari platform DeepAI. Skrip ini membongkar algoritma enkripsi klien *DeepAI*, sehingga Anda dapat menggunakan layanan Generator Gambar *High-Definition* dan mengobrol dengan model AI tingkat Premium secara **gratis tanpa batas (Bypass API-Key)**.

Skrip ini memiliki dua wujud mematikan:
1. 💻 **CLI Interaktif** yang elegan dengan efek mesin tik dan *spinner* animasi.
2. 🌐 **REST API Server** untuk mengintegrasikan model-model canggih ini ke aplikasi atau *bot* milik Anda sendiri.

## 🚀 Panduan Penggunaan

### 💻 1. Mode CLI Interaktif
Cukup panggil skrip tanpa argumen, dan Anda akan disambut oleh UI elegan.
```bash
node dp.js
```
*Kemudian pilih Modul 1 (Image), Modul 2 (Server API), atau Modul 3 (Chat).*

**Trik Khusus Chat [Vision] 📸:**
Saat berada di dalam Mode Chat, jika Anda memilih model yang mendukung **Vision**, Anda bisa menyuruh AI menganalisis gambar lokal di PC Anda. Awali *prompt* dengan `[image]` dan masukkan lokasi gambar:
```text
USER  > [image] "C:\Users\Saya\Pictures\gambar.png" Tolong jelaskan isi foto ini!
```
*Skrip otomatis akan mengunggah gambar tersebut secara anonim dan menyuntikkannya ke memori AI!*

<br>

### 🌐 2. Mode REST API Server
Jalankan NEXUS AI di latar belakang untuk melayani *request* dari aplikasi lain.
```bash
node dp.js --api
```
Server akan berjalan di `http://localhost:3000`.

#### A. Endpoint Image Generator (POST `/api/generate`)
```json
{
  "prompt": "Cyberpunk city at night, neon lights, 4k",
  "isQuality": true,
  "style": "hd",
  "negativePrompt": "blurry, distorted"
}
```

#### B. Endpoint Chat + Image (POST `/api/chat`)
*(Pastikan Anda telah mengunggah gambar terlebih dahulu jika butuh vision)*
```json
{
  "model": "gemma-4",
  "messages": [
    { 
      "role": "user", 
      "content": "Jelaskan gambar ini", 
      "attachment_uuids": ["df99118c-8bdb-4bd9-8bb5-f783e27db59a"]
    }
  ]
}
```
